# Sommelier (Prototype) — Erasmus BIP 2025

This repository contains the group project developed during the Erasmus BIP “Heritage in Video Games” (Mirandela, Portugal, 2025).

Sommelier is a prototype horror game inspired by the cultural heritage surrounding sommeliers and wine tasting.  
The player moves with WASD and the goal is to survive four nights.

## Project context
- **Type:** Group project / prototype
- **Team:** 6 international students from 4 different countries
- **Engine:** Unity (C#)

## My contribution
I was involved in:
- Initial concept and gameplay direction
- Programming the core game systems and overall game flow/state management in Unity
- Implementing the main menu and related UI logic

## How to play
- Movement: **WASD**
- Objective: **Survive 4 nights**
